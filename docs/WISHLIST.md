# OrdexWallet Feature Wishlist

Long-term features that are not currently planned for implementation:

## Hardware Wallet Integration
- Support for hardware wallet devices (Ledger, Trezor, etc.)
- Secure private key storage on hardware devices
- Transaction signing via hardware wallet
- Requires significant development and testing

## Multi-Signature Wallet Support
- M-of-N multi-signature wallets
- Multiple parties required to approve transactions
- Enhanced security for high-value wallets
- Requires changes to wallet creation and transaction signing

## Lightning Network Integration
- Payment channels for instant, low-fee transactions
- Built on top of OrdexCoin (OXC) as the base layer
- Requires network-level support from OrdexCoin
- Future feature pending OXC protocol development

---

*These features are marked as wishlist - they represent desired functionality but are not currently in the development roadmap.*
