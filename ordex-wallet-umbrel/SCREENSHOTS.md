# Umbrel App Screenshots Guide

## ✅ Official Umbrel Requirements:

| Item | Specification |
|---|---|
| **Number of screenshots** | 3 to 5 screenshots |
| **Resolution** | 1440px × 900px |
| **Format** | PNG or JPG |
| **Location** | Place directly inside `/ordex-wallet-umbrel/` folder |
| **Naming** | `screenshot-1.png`, `screenshot-2.png`, `screenshot-3.png` |

---

## 📋 Recommended Screenshots:

1.  **Dashboard** - Main wallet overview showing balances and sync status
2.  **Send Transaction** - Send page with form fields
3.  **Transaction History** - Transactions list view

---

## 🔄 Update umbrel-app.yml:

Add to the gallery array in `/ordex-wallet-umbrel/umbrel-app.yml`:

```yaml
gallery:
  - screenshot-1.png
  - screenshot-2.png
  - screenshot-3.png
```

✅ **Note**: Screenshots are optional but greatly increase chances of app being accepted into the official store.
