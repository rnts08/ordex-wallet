About Ordex Web Wallet

Project Overview

The Ordex Web Wallet is the official, self-hosted custodial solution for the Ordex Network. Designed for maximum security and ease of use, it allows you to manage your OrdexCoin (OXC) and OrdexGold (OXG) assets in a completely private environment.

This application provides a unified interface for chain interactions, transaction management, and administrative oversight, ensuring that community members can maintain full control over their financial sovereignty within the Ordex ecosystem.

Support the Project

Ordex is a community-driven open-source project. Your support helps us fund core development, maintain network infrastructure, and refine the protocol through rigorous testing.
